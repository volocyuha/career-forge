# Interview Prep — STAR Stories

Stories built from real project evidence in this repository.
Generated or updated by `/interview [role]`. Add your own manually at any time.

Each story answers a specific interview question using the STAR format,
grounded in wiki evidence so you can speak to it confidently.

---

## How to Use This File

1. Run `/interview [role]` to generate a set of stories for a specific role
2. Read each story and adjust the language to sound like you
3. Practice speaking each story out loud — it should take 90-120 seconds
4. Add the Evidence field to every story so you know what to reference if probed

---

## STAR Story Template

Copy this block for each new story:

```
### [STORY TITLE]

**Answers:** [Interview question this story addresses]

**Situation:** [Context — what project, what stage, what constraints]

**Task:** [What you were responsible for — your specific scope]

**Action:** [What you did — specific decisions, not generic activities]

**Result:** [What changed — numbers preferred, observable change acceptable]

**Evidence:** [What you can point to: case study, portfolio page, artifact, metric]
```
