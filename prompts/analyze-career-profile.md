# Prompt: Analyze Career Profile

Use this prompt to get a structured analysis of your current career position.
Paste it into your AI assistant after filling in the repository modules.

---

```
Read the following files from this repository:
- career-profile/profile.md
- career-profile/current-role.md
- career-profile/skills.md

Based only on the information in these files:

1. Summarize my current career position in 2-3 sentences
2. List my strongest evidence (High or High+ skills with specific project proof)
3. List my most significant gaps relative to my stated target role
4. Identify any inconsistencies between what I claim and what the evidence shows
5. Rate my portfolio readiness on a scale: Not ready / Needs work / Ready for junior roles / Ready for target role

Do not invent experience. If information is missing, say so explicitly.
Mark any uncertain conclusions as [Needs confirmation].
```
