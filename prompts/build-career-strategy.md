# Prompt: Build Career Strategy

Use this prompt to generate or update your career strategy.
Run after completing your gap analysis.

---

```
Read the following files from this repository:
- career-profile/profile.md
- career-profile/current-role.md
- career-profile/target-role.md
- gap-analysis/[your-gap-analysis].md
- career-profile/skills.md

Generate a career strategy document with:

1. Current position summary (2-3 sentences, evidence-based)
2. Target position and realistic timeline given current gaps
3. Strategic direction — plain language description of the path from current to target
4. Next best projects — ordered by strategic impact, with scope and portfolio output defined
5. Career positioning statement — 1-2 sentences for CV, LinkedIn, and cover letters
6. Application strategy — where to apply, in what order, and when based on gap closure

Rules:
- Base all recommendations on actual evidence in the repository
- Do not recommend applying to roles before the critical gaps are closed
- Be direct about what is missing — strategic clarity is more useful than encouragement
- Mark any assumption as [Assumption: needs confirmation]

Save output to: career-profile/strategy.md
```
